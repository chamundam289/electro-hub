# Popup Setup Instructions

## How to Enable Popup on User Side:

### Step 1: Admin Login
1. Go to `/admin` and login with admin credentials
2. Navigate to **Website Settings**

### Step 2: Configure Popup
1. Go to **Popup** tab
2. Turn ON "Enable Popup Banner"
3. Add popup image URL (example: `https://images.unsplash.com/photo-1607083206869-4c7672e72a8a?w=400&h=300&fit=crop`)

### Step 3: Configure WhatsApp
1. Go to **WhatsApp** tab  
2. Set your WhatsApp number (with country code, e.g., `+911234567890`)
3. Customize "Popup WhatsApp Message" template
4. Click **Save Settings**

### Step 4: Test on User Side
1. Open website in new browser tab/incognito mode
2. Wait 3 seconds - popup will appear at bottom center
3. Popup contains: Image + WhatsApp Button + Later Button
4. Click WhatsApp button to test message sending

## Popup Features:
- ✅ Shows at bottom center after 3 seconds
- ✅ Simple design: Image + 2 buttons only
- ✅ Shows once per browser session
- ✅ Fully controlled from admin panel
- ✅ WhatsApp integration with custom messages
- ✅ Mobile responsive

## Requirements:
- Popup enabled in admin settings
- Image URL provided
- WhatsApp number configured
- User hasn't seen popup in current session

## Admin Control:
- Enable/disable popup anytime
- Change image dynamically  
- Customize WhatsApp message
- Real-time preview in admin panel
